# admission-wedge — docs

The **live-memory admission wedge**: design and gate history for the initiative
that follows recall-wedge (admission of memory after recall). Implementation-adjacent
code lives in `packages/persona-engine/src/recall-wedge/admission-wedge-*`.

**Start here:** [`ADMISSION-WEDGE-MVP-DESIGN.md`](ADMISSION-WEDGE-MVP-DESIGN.md).

The remaining files are **Phase gate / acceptance / contract-reconciliation** docs
from the build cycle, cited as `Authority: docs/admission-wedge/…` provenance in
the code they governed. `dixie-probes/` and `fixtures/` hold the probe payloads
and seeded fixtures used by the reducer/runner tests.
